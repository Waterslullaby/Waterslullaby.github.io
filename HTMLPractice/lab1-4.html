Common HTML Tags

These are the most commonly used HTML tags and a short description of each.

Block elements

Block elements take up their own line and take up the full width of the page.

<body>

Lets you create and edit a body.

<head>

Element that groups together all the elements that would go in the head. Like title, style, base, link, meta, script and noscript.

<html>

Element that tells the browser that the file is HTML.

<div>

Seperates a portion of a document. Used more for styling purposes over section.

<section>

Groups elements inside as a generic selection.

<header>

Element typically used to group all the intro and navigational items.

<main>

Element used for content that is specific to the document.

<article>

Element used for standalone content.

<nav>

Element used to make a set of navigational links.

<footer>

Element used as the footer of the document. Usually where things like authorship, copyright, contact information go. Site maps, back to top links, and related documents can go here also.

<p>

Adds space before and after the elements in between the tags.

<aside>

Element used to seperate content from the content it was placed in but should stil be related to the original content.

<li>

Creates a list.

Inline elements

Inline elements only take up as much space on a line as they need to. They do not start new lines.

<a>

This tag defines a hyperlink.

<strong>

Bolds the text inbetween the tags.

<em>

Adds emphasise to the text inbetween the tags. Looks like italics.

<img>

Element that allows you to use images.

<span>

Element used to group inline elements, usually used to add a hook. The tag however does nothing by itself so has to be used with things like style = ""
